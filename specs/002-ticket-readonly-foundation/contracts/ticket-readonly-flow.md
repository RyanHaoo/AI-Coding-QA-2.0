# 契约: 工单只读浏览流程

## 范围

本契约描述阶段 2 的页面状态、查询参数、数据范围和可观察行为。它不是外部 API 规范；阶段 2 不新增公开 API route。

## 页面入口

| 入口 | 当前角色 | 页面意图 |
|------|----------|----------|
| `/?view=tickets` | 质检员、施工方 | 当前身份相关工单列表 |
| `/?view=admin-tickets` | 管理员 | 当前项目全部工单基础列表 |
| `/?view=tickets&ticketId={id}` | 质检员、施工方 | 从普通列表打开只读详情 |
| `/?view=admin-tickets&ticketId={id}` | 管理员 | 从管理员列表打开只读详情 |

## 查询参数

| 参数 | 可选值 | 默认值 | 说明 |
|------|--------|--------|------|
| `view` | `tickets`、`admin-tickets` | 阶段 1 默认规则 | 决定普通列表或管理员基础列表 |
| `ticketStatus` | `pending`、`closed`、`all` | `pending` | 普通工单列表状态筛选 |
| `ticketSort` | `newest`、`oldest` | `newest` | 普通工单列表排序 |
| `ticketId` | 工单 id | 无 | 存在时打开只读详情 |

无效 `ticketStatus` 或 `ticketSort` 值按默认值处理。无效 `ticketId` 按未找到处理。

## 普通工单列表契约

### 质检员

**Given** 当前身份角色为质检员  
**When** 用户访问 `/?view=tickets`  
**Then** 列表只包含当前项目内由当前身份发起的工单。

### 施工方

**Given** 当前身份角色为施工方  
**When** 用户访问 `/?view=tickets`  
**Then** 列表只包含当前项目内当前责任人为当前身份的工单。

### 列表项展示

每个列表项必须展示：

- 工单编号
- 状态
- 严重程度
- 问题描述摘要
- 详细位置
- 创建时间
- 当前责任人
- 专业类型
- 如有图片，展示一张现场图片预览或图片存在提示

### 筛选与排序

- `pending` 只展示待处理。
- `closed` 展示已完成和已拒绝。
- `all` 展示当前角色固定范围内全部状态。
- `newest` 表示同组内创建时间倒序。
- `oldest` 表示同组内创建时间正序。
- 紧急工单始终排在非紧急工单之前。

## 管理员基础列表契约

**Given** 当前身份角色为管理员  
**When** 用户访问 `/?view=admin-tickets`  
**Then** 列表展示当前项目全部工单，不按发起人或责任人限制。

管理员列表项必须展示：

- 工单编号
- 当前状态
- 严重程度
- 专业类型
- 问题描述摘要
- 创建时间
- 当前责任人
- 发起人
- 查看详情入口

阶段 2 管理员列表不要求完整筛选组合；完整管理员工单中心进入后续阶段。

## 只读详情契约

**Given** 用户有权访问某张工单  
**When** 用户通过 `ticketId` 打开详情  
**Then** 详情按以下分区展示。

### 基础信息

- 工单编号
- 状态
- 严重程度
- 创建时间
- 发起人
- 所属项目
- 当前责任人

### 问题信息

- 专业类型
- 问题描述
- 详细位置
- 现场图片
- 问题详情

### 处理信息

- 问题归因
- 预防建议

缺失内容以“未填写”或“暂无”展示。

### 处理记录

- 操作类型
- 操作人
- 操作时间
- 操作原因
- 操作内容

处理记录按时间倒序展示。

## 异常状态契约

### 空列表

**Given** 当前身份没有符合筛选条件的工单  
**When** 用户查看列表结果区  
**Then** 展示“当前范围暂无工单”或“当前筛选无结果”。

### 无权限

**Given** 用户访问自己无权查看的工单  
**When** 系统完成权限判断  
**Then** 展示无权限提示，不展示工单字段、协作人姓名或处理记录。

### 未找到

**Given** 用户访问不存在的工单 id  
**When** 系统完成查询  
**Then** 展示未找到提示，并提供返回当前列表入口。

### 身份失效

**Given** 当前身份 cookie 对应的项目身份不存在或已失效  
**When** 用户访问工单页面  
**Then** 阻止读取工单数据，并引导用户重新登录或选择身份。
